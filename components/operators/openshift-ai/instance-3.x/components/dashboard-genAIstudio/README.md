# dashboard-genAIstudio

## Purpose
This component is designed enable Gen AI Studio.

## Usage

This component can be added to a base by adding the `components` section to your overlay `kustomization.yaml` file:

```
apiVersion: kustomize.config.k8s.io/v1beta1
kind: Kustomization

resources:
  - ../../base

components:
  - ../../components/dashboard-genAIstudio
```
